---
description: "Prepare a release"
---
Update the `CHANGELOG.md` file (create one if it does not exist) by doing the following:

1. Compare the differences between the last commit of the `main`/`master` branch with the `dev`/`feature`/`bugfix`/`release` branch.
2. Summarize the differences into the following sections.
2a. New
2b. Changed
2c. Removed
2d. Bugfixes
and add them to the `CHANGELOG.md`
