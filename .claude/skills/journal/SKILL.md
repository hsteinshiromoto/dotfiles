Do the following:
1. Summarize the conversation with the user in this session: add the initial user's request, your reasoning, the user accepted solution, and results of the `/sanity_check`.
2. Summarize the changes that you have done to any of the files available to you.
3. Describe any relationships that you've used across the same or different files. For example, when coming up with a solution, did you had to look at other files to understand the context/contrained of the solution? If yes, then what were these files, what did you search/find in them, and why?
4. Update (or create, if the note does not exist) a note `journal/YYYY-DD-DD.md`, and update (or create, if it does not exist) `journal/summary.md`.
